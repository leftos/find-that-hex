Find That Hex
	by Lefteris "Leftos" Aslanoglou

	
Quick Notes
	This tool is meant as a way to quickly find non in-byte-aligned hex data in a file.

	You can either simply run it and enter the required information, or run it via command-line or shortcut as follows:
	FindThatHex.exe <path> <string> <start_offset>
